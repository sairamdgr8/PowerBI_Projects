In this article, we are going to discuss a Leet code question type senario.

Sai has a phone number of 10 digits with the sum of its digits missing.

Find the missing digits in the number are represented by a question mark(?).But Sai remembers that the given phone number possible divisible by 3

Sample input 5?9?3????? Sample Output 5090300001 Explanation If we replace question marks in the given numbers 5?9?3????? with single-digit numbers, 5090300001 is the minimum possible number divisible by 2 Hence the output is 5190300000

def find_min_missing_number(s):
    count=0
    for i in list(s):
        if i!="?":
            count=count+int(i)
    for value in range(0,count):
        if (count+value)%3==0:
            break
    list_s=list(s)
    list_index=list_s.index("?")
    list_s[list_index]=str(value)
    return(int("".join([(i) if i!='?'  else '0'  for i in list_s ])))

#check

find_min_missing_number("5?9?3?????")
>>> 5190300000
